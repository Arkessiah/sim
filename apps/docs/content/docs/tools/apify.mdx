## Usage Instructions

Execute Apify Actors for web scraping, data extraction, and browser automation. Choose between synchronous runs (quick results) or asynchronous runs (for longer tasks with polling).



## Tools

### `apify_sync_run`

Run an Apify Actor synchronously and return its output (300s timeout)

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `actorId` | string | Yes | Apify Actor ID or name \(e.g., web-scraper or apify/web-scraper\) |
| `apiKey` | string | Yes | Apify API token for authentication |
| `input` | object | No | Input data to pass to the actor \(JSON object\) |
| `timeout` | number | No | Timeout in seconds \(max 300 for sync runs\) |
| `maxItems` | number | No | Maximum number of results to return |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `data` | json | The output data from the Actor run |
| `runId` | string | The ID of the Actor run |
| `actorId` | string | The ID of the Actor that was run |
| `status` | string | Final status of the Actor run |
| `stats` | json | Runtime statistics \(memory, CPU usage, etc.\) |
| `usage` | json | Resource usage details \(compute units, data transfers, etc.\) |

### `apify_sync_get`

| --------- | ---- | -------- | ----------- |

| `apiKey` | string | Yes | Apify API token for authentication |
| `timeout` | number | No | Timeout in seconds \(max 300 for sync runs\) |
| `maxItems` | number | No | Maximum number of results to return |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `data` | json | The output data from the Actor run |
| `runId` | string | The ID of the Actor run |
| `actorId` | string | The ID of the Actor that was run |
| `status` | string | Final status of the Actor run |
| `stats` | json | Runtime statistics \(memory, CPU usage, etc.\) |
| `usage` | json | Resource usage details \(compute units, data transfers, etc.\) |

### `apify_async_run`

Run an Apify Actor asynchronously and poll for completion (30min max)

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `actorId` | string | Yes | Apify Actor ID or name \(e.g., web-scraper or apify/web-scraper\) |
| `apiKey` | string | Yes | Apify API token for authentication |
| `input` | object | No | Input data to pass to the actor \(JSON object\) |
| `timeout` | number | No | Timeout in seconds for the actor run |
| `maxItems` | number | No | Maximum number of results to return |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `data` | json | The output data from the Actor run |
| `runId` | string | The ID of the Actor run |
| `actorId` | string | The ID of the Actor that was run |
| `status` | string | Final status of the Actor run |
| `defaultDatasetId` | string | ID of the default dataset containing results |
| `stats` | json | Runtime statistics \(memory, CPU usage, etc.\) |
| `usage` | json | Resource usage details \(compute units, data transfers, etc.\) |
